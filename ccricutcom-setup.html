Cricut.com/setup – Download & Install

For beginners, access the online manual for your Cricut machine via cricut.com/setup. Setting up a new Cricut machine can be exciting, but sometimes it’s a little intimidating. If you require to set up a new Cricut machine, you have come to the right place. Here, we will show you the steps to complete the setup process to move on to the fun part. Let’s get started with cricut.com/setup to create your first project.


What Type of Components Do You Get with a New Cricut Maker?

Including the color choice, you get a few options for bundles while buying the new maker from the official Cricut store. If you were to buy Cricut Maker Machine (any color), here is what you will find while unboxing:


  • As usual, the first thing will be your Cricut Machine.
  • A USB cable and power cord for your machine to connect with the desktop.
  • And a mat with the measurement of 2-12” x 12”, one blue light grip, and one pink fabric grip.
  • The most important part is the “Rotary Blade.”
  • Black Fine Point Pen
  • A few small material pieces for your practice project
  • Premium Fine Point Blade with housing
  • A welcome book for Cricut Maker

How to Set Up a Cricut Machine to Make the First Cut?

You will need to use Cricut properly and set it up first to cut different materials. Here is how to set up your machine with ease.


  1. Start with choosing your material.
  2. Then, take out the sticky mat that came with the product packaging.
  3. Visit cricut.com/setup and download the app location Cricut Setup.
  4. Choose a design for your craft project in Cricut.
  5. Select the settings for the material you require.
  6. Now, send your craft designs into the machine.
  7. Tap the button on the Cricut machine.
  8. Once the project completes, you can remove the mat and the material.
  9. Now, you know how to make the first cut by setting up the machine correctly.

Download & Install the Cricut.com/setup on Your Device for Mac

  1. Open an internet browser and go to the official website cricut.com/setup.
  2. Then, select the option Download.
  3. Next, your screen and the browser will change.
  4. Now, you need to double-click the (.dmg) file in the browser or the Downloads folder.
  5. When the downloading process is complete, follow the on-screen prompts.
  6. After that, pull the Cricut Setup icon to the app folder icon.
  7. And start installing the software.
  8. Thereafter, an icon Cricut for your desktop will automatically be added to the app folder.
  9. To launch the Setup Cricut for your desktop, double-click the Cricut Setup in the application folder.
  10. To use the shortcut, drag the app icon to the dock.
  11. A mac notice will prompt you to verify if you need to open the downloaded app.
  12. If yes, click the Open button and continue the process.
  13. Lastly, sign in with a Create a Cricut ID and password for Cricut Software.
  14. Begin your first design cut with Cricut Setup for your Mac device

How to Pair Up a Printer to the Cricut Model?

Well, you do not necessarily need a printer to cut materials with Cricut. But if you need to print designs onto fabric or any other material, you can set up your home printer with the following instructions.


  1. Start with downloading the Cricut software or app through cricut.com/setup.
  2. Then, sign in to the Cricut account.
  3. Register an id if this is your first time using the Cricut setup.
  4. Next, select your Cricut device and click on the Download button.
  5. After that, let it complete the setup.
  6. Now, double-click on the install .exe file to finish the installation.
  7. You can create new designs with Cricut now.
  8. Next, turn on the machine and connect it to the desktop via Bluetooth or USB.
  9. In the end, your Cricut is set through cricut.com/setup to process the design via a printer.

How to Set Up Cricut on a Printer Through Your Smartphone?

Connecting the Cricut machine to the printer using your smartphone is quite easy. Follow the steps to perform the procedure.

  1. Start with assembling the Cricut machine and turn it on.
  2. Then, use the Bluetooth option to connect Cricut with mobile.
  3. You have to download the Cricut application and launch it using the App Store.
  4. Next, sign in to the app with the id you created via cricut.com/setup for Cricut.
  5. Tap on the menu and choose your device.
  6. Now, follow the on-screen instructions and create desired projects.
